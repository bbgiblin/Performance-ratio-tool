# Performance-ratio-tool
Weighted performance metrics calculator

WORKFLOW EFFICIENCY INDEX TOOL - QUICK START

OPTION 1: One-Click Setup (Windows)
   Double-click INSTALL_AND_RUN.bat

OPTION 2: Manual Setup (All IDEs)
   1. Open this folder in VS Code, PyCharm, or any Python IDE
   2. Open terminal
   3. Install dependencies:
      pip install -r requirements.txt
   4. Run the tool:
      python -m streamlit run workflow_efficiency_ui.py

Browser will open automatically at http://localhost:8501

To stop: Press Ctrl+C in the terminal

IMPORTANT: Always use "python -m streamlit run" (not just "streamlit run")
